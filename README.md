# ⚡ NEWSEED v-1-0-0

> **Официальный релиз ультимативного обходчика.** > Забудь про лаги в YouTube, Discord и вечный VPN для ChatGPT.

---

### 🚀 Что умеет NEWSEED?
* **YouTube & Discord:** Полный обход замедлений, работа видео в 4K и стабильный голос в дискорде.
* **AI UNBLOCK:** Разблокировка ChatGPT, Claude и Gemini одной кнопкой (без VPN).
* **Auto-Admin:** Скрипт сам запрашивает права администратора.
* **ISP Detect:** Автоматическая настройка под твоего провайдера (МТС, Ростелеком, Билайн, Дом.ру и др.).

---

### 🛠 Установка и запуск
1. **Скачай архив** с релизом.
2. **Распакуй** в любую папку (главное — не в системную, чтобы не было проблем с правами).
3. Запусти файл `▶️ START_NewSeed.bat`.
4. В меню выбери **[1]** для старта или **[3]** для разблокировки ИИ.

---

### 🎨 Особенности сборки
* **Технологичный дизайн:** Минимум эмодзи, только серьезный CLI-интерфейс.
* **Оптимизация:** Автоматическая очистка кэша DNS и настройка сетевого стека.
* **Чистота:** Никаких лишних файлов, всё лежит по папкам `bin` и `ssp`.
